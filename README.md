## Install
```shell
xxh +I https://github.com/xxh/xxh-plugin-prerun-zoxide
```

## Usage
```shell
xxh host
cd /etc
cd /tmp
z e
# /etc
```

## Supported xxh-shells

* xxh-shell-xonsh
* xxh-shell-zsh
* xxh-shell-fish
* xxh-shell-bash-zero
